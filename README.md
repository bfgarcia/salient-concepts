### About

Salient Concepts LLC builds privacy-focused smart home technology. We believe your home data should stay in your home — not on someone else's server. Our products run entirely on locally hosted infrastructure with zero cloud dependencies.

### Products

**EPOCH** — A production-grade AI nervous system platform. 7-layer architecture, 85 MCP tools, constitutional governance, 3-tier intelligence routing. Powers voice assistants, home automation, telephony, robotics, and security intelligence from hardware you own.

**Aria** — An AI voice assistant that runs on EPOCH. Natural language voice control, speaker and face recognition, phone calls and SMS, music control, and smart home integration. Sub-2-second response time. Your voice data never leaves your network.

### Technology Philosophy

- **Privacy-first** — All processing on local GPUs. No cloud APIs, no external data transmission.
- **Production-grade** — Not a prototype. Running 24/7 in production with constitutional governance.
- **Open standards** — Built on MCP (Model Context Protocol), Ollama, Home Assistant, Asterisk, and PostgreSQL.
